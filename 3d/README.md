# Floria Link 3D

Godot 4とBlender MCPを使用する3D版です。

## フォルダ

- `blender/` — マップ制作、画像資料、Blendファイル、再生成スクリプト
- `godot/` — Godotゲーム本体

Blenderの完成マップは `blender/floria_awakening_garden_gameplay.blend`、
ゲームカメラの確認画像は `blender/floria_awakening_garden_combat_preview.png` です。

次の作業ではBlenderマップをGLBへ書き出し、`godot/assets/environment/`へ配置します。
