# 赤キャラクター：3D投影テクスチャ用画像生成プロンプト

## 使用方法

元の赤キャラクター画像を添付し、以下のプロンプトをそのままChatGPTへ送ってください。
これは通常のキャラクター設定画ではなく、Blenderで正面・側面・背面投影を行うための画像です。

## 正式プロンプト

```text
添付した赤キャラクターを、Blenderで3Dモデルへ直接投影してテクスチャとして使用するための、正投影4方向ターンアラウンド画像として再生成してください。

これはデザイン案やポーズ集ではありません。画像のピクセルを3Dモデル表面へ直接投影するため、以下の条件を厳密に守ってください。

【キャラクターデザイン】

添付画像のデザインを変更せず、以下を完全に維持してください。

・赤い髪と高いポニーテール
・左右の長い尖り耳
・琥珀色の瞳
・髪に付いた白、橙、赤の花飾り
・胸元の白い花飾り
・赤と深い焦げ茶を基調にしたノースリーブのコルセット
・金色の縁取り、留め具、装飾
・肩から分離した赤い花弁状の袖
・暗色の手首装飾
・腰から広がる複数層の赤い花弁スカート
・スカート内側の焦げ茶色のショートパンツ
・太腿の小さな蔦と花飾り
・赤い花弁装飾が付いたブーツ
・添付画像と同じ体格、頭身、顔立ち、脚の長さ

新しい装飾を追加しないでください。既存の装飾を省略しないでください。

【配置する4方向】

1枚の正方形キャンバスに、次の4方向を左から右へ横一列で配置してください。

1. 完全な正面
2. 完全な右側面
3. 完全な背面
4. 完全な左側面

斜め向きや3/4ビューにはしないでください。すべて完全な正投影にしてください。

【姿勢と整合性】

・全方向で同一人物、同一衣装、同一髪型にする
・全方向で身長、頭身、頭の大きさ、肩幅、腰幅、脚の長さを完全に一致させる
・全方向で髪、花、金具、衣装の縫い目、花弁の枚数と位置を一致させる
・左右側面と背面は、正面と矛盾しない構造にする
・ニュートラルなAポーズにする
・両腕を胴体から約30度離す
・手と指を衣装や身体に重ねない
・両脚を軽く離し、左右の脚が重ならないようにする
・顔は穏やかな無表情、口は閉じる
・全方向で足裏の高さを同じ水平線に揃える
・全方向でキャラクターの中心線と表示倍率を揃える
・帽子、武器、杖、炎、魔法、花びら、モーション表現は追加しない

【背景と透明度：最重要】

・背景は実際のアルファチャンネルを持つ完全透過にする
・白背景、灰色背景、色付き背景を使用しない
・透明を表す市松模様を画像として描かない
・キャラクター以外の領域はRGBAのAlphaを0にする
・キャラクターの輪郭に白い縁、白いマット、白いフリンジを残さない
・髪の隙間、腕と胴体の隙間、脚の間も正しく透明にする
・床、地面、接地影、ドロップシャドウ、反射を描かない
・発光、煙、粒子、エフェクトを描かない

【画像品質と出力】

・可能な限り4096×4096ピクセル
・正方形、1:1
・PNG形式
・実アルファチャンネル付きRGBA
・最高品質、輪郭と衣装模様が明瞭
・均一で柔らかい正面照明
・方向ごとに明るさや色を変えない
・パース、広角歪み、被写界深度を使用しない
・各ビューの間に十分な透明余白を設け、互いに重ねない
・キャラクター全体を切らず、髪の先端から靴底まで収める
・文字、方向名、罫線、寸法線、ロゴ、透かしを入れない

最終出力前に、次の項目を自己確認してください。

・4方向の身長と足元が一致している
・正面と背面の左右幅が一致している
・両側面の奥行きと輪郭が一致している
・ポニーテール、耳、花飾り、胸飾り、スカート、ブーツの位置が全方向で矛盾していない
・背景が見た目だけでなく、実際に透明である
・白い背景ピクセルや輪郭の白フリンジが残っていない

条件を満たした1枚のターンアラウンド画像だけを出力してください。
```

## 生成後の確認事項

画像を保存したら、次を確認してください。

- ファイル形式が `.png` である
- 画像ビューアーで背景が透明として表示される
- 白い四角形の背景が画像データに含まれていない
- 4方向の足元と頭頂が揃っている
- 正面、側面、背面で装飾の数と位置が一致している
- キャラクター同士が重なっていない

背景が白く出力された場合は、その画像を添付して次の修正指示を送ってください。

```text
キャラクターの形、色、配置、4方向の整合性は一切変更せず、背景だけを実アルファチャンネル付きの完全透過へ変更してください。
白背景を透明に見せかけるのではなく、キャラクター以外の全ピクセルのAlphaを0にしてください。
輪郭の白いフリンジ、床の影、市松模様、文字は残さないでください。
PNG形式で出力してください。
```
